# Implementing-a-new-programming-language: Fika
To run Fika, both [Java](https://www.java.com/en/download/) and [LLVM's llc](https://llvm.org/docs/GettingStarted.html) need to be installed. 

## Extension installation
The latest "stable" version of the extension is avalliable in the root repository as ``fika-x.y.z.vsix`` and can be installed through the *Extensions* tab on VSCode, extending the context menu (top right ``...``) and then selecting ``Install from VSIX``
